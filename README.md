## DSPy

- Modular prompts and pipelines for LLMs
- Deterministic System prompt
- Works with different LLMs out of the box
- Prompt Optimizers requires datasets and metrics to optimize the prompt

### Prompt Optimizers:

    1. GEPA
    2. MIPRO v2

## Mlflow

- Mlflow is similar to **Langsmith** with eval and tracing
- Meta Prompt Optimizer is a zero shot prompt which can enrich system prompt based in feedback

Run mlflow by this command

```
mlflow server
```
